We're excited to introduce **Minions Audiences** — Ideal customer profiles, niche definitions, market segments, and target audience personas.

- **TypeScript SDK**: `npm install @minions-audiences/sdk`
- **Python SDK**: `pip install minions-audiences`
- **CLI**: `npm install -g @minions-audiences/cli`
