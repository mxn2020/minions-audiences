# Minions Audiences

Ideal customer profiles, niche definitions, market segments, and target audience personas

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-audiences/sdk`) — Core library for Node.js
- **Python SDK** (`minions-audiences`) — Python bindings
- **CLI Tool** (`@minions-audiences/cli`) — Command-line interface
- **Web App** — Interactive playground at [audiences.minions.wtf](https://audiences.minions.wtf)
