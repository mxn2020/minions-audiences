---
title: Installation
description: Install Minions Audiences packages
---

## TypeScript / Node.js

```bash
npm install @minions-audiences/sdk minions-sdk
```

## Python

```bash
pip install minions-audiences
```

## CLI

```bash
npm install -g @minions-audiences/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
